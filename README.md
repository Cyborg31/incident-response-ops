# Incident Response & Operations

This repository contains notes and experiments simulating real-world operational incidents in systems and cloud-based applications.

The focus is on observing failures, investigating root causes, and documenting outcomes. These exercises are part of hands-on learning in troubleshooting, monitoring, and system operations.

## Topics covered
- Simulated service outages
- Monitoring and alert observations
- Troubleshooting and root cause reasoning
- Documenting outcomes and lessons learned

## Tools & Environment
- Linux and Windows environments
- Local applications / Docker containers
- Logs, alerts, and metrics monitoring
- Azure infrastructure
